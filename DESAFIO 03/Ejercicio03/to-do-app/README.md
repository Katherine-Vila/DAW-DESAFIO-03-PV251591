# Aplicación de Lista de Tareas (To-Do App) en Angular

¡Hola! Este es el **Ejercicio 03** del Desafío 03 para la materia de Desarrollo de Aplicaciones Web (DAW404) de la Universidad Don Bosco.

## ¿De qué trata este proyecto?
Es un CRUD completo de una lista de tareas construido con **Angular**. Permite agregar nuevas tareas indicando su nombre, descripción y nivel de prioridad. También cuenta con la funcionalidad de editar, eliminar y un sistema de paginación para organizar las tareas largas. Todo el diseño de la interfaz fue hecho utilizando **Bootstrap 5**.

## Cómo ejecutar el proyecto localmente

Si quieres probar el proyecto en tu propia computadora, sigue estos sencillos pasos:

1. Abre tu consola o terminal y navega hasta esta carpeta.
2. Instala las dependencias necesarias escribiendo:
   ```bash
   npm install
   ```
3. Una vez termine, levanta el servidor local con:
   ```bash
   npm start
   ```
4. Abre tu navegador web favorito y entra a `http://localhost:4200`. ¡Y listo!

## Tecnologías utilizadas
* **Angular CLI** (versión 16)
* **TypeScript**
* **Bootstrap 5** (Para el diseño y la interfaz responsiva)
