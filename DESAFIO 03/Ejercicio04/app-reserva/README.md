# Cotizador de Agencia de Viajes en React

¡Hola! Este es el **Ejercicio 04** del Desafío 03 para la materia de Desarrollo de Aplicaciones Web (DAW404) de la Universidad Don Bosco.

## ¿De qué trata este proyecto?
Es un cotizador interactivo en tiempo real construido con **React** y el framework **Next.js**. 
La aplicación permite simular el costo total de un paquete de viajes basado en la cantidad de adultos y niños, el tipo de hotel, el plan de alimentación en restaurante y el tipo de vuelo. Toda la lógica matemática se ejecuta instantáneamente a medida que el usuario interactúa, aprovechando el manejo de estados (`useState`) de React.

## Cómo ejecutar el proyecto localmente

Para echar a andar la aplicación en tu computadora, sigue estos pasos:

1. Abre la terminal y sitúate dentro de esta carpeta.
2. Descarga los paquetes necesarios ejecutando:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo de React escribiendo:
   ```bash
   npm run dev
   ```
4. Abre tu navegador web y dirígete a `http://localhost:3000`. ¡Ahí verás la agencia funcionando!

## Tecnologías utilizadas
* **React** (Para el manejo de estados e interactividad)
* **Next.js 15** (Framework principal)
* **Bootstrap 5** (Para el diseño visual y adaptación a pantallas)
